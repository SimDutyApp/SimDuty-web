Overview

Use this page to resolve startup, plugin, and telemetry issues in a deterministic order.

Estimated time: 5 minutes Focus: SDK/NET recovery Result: restored connection

Plugin not detected

  • SDK FAIL: verify scs-telemetry.dll in \bin\win_x64\plugins.
  • NET FAIL: verify simduty.dll in \bin\win_x64\plugins.
  • Confirm both DLL files are in the correct game directory.
  • Confirm the game is launched in x64 mode with matching plugin files.
  • Restart in this sequence: SimDuty first, then ATS/ETS2.

No truck connection

  • Launch SimDuty before opening ATS/ETS2.
  • Enter the truck cabin to initialize telemetry reads.
  • Review permission, antivirus, and firewall restrictions.

Quick FAQ

  • Can I use it offline? Yes, for local operation.
  • Do I need to install .NET? No, current releases are self-contained.
  • Does it work in ETS2 and ATS? Yes, with correct plugins.

Escalation

If all checks pass and SDK/NET still fail, collect screenshots, plugin paths, and app status, then open a support request in Discord.
